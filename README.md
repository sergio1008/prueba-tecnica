# API para la gestión de Naves espaciales 

  This API provides a set of endpoints for CRUD management of Spaceships.

### Specifics
  Java 17


## API Documentation
- http://localhost:9090/swagger-ui/index.html

## Getting Started

- mvn clean verify
- mvn spring-boot:run


### Configuration

- Default port: 9090
